# Lichtblick Betreuung

Moderne, responsive Website für Lichtblick Betreuung. Das Projekt ist bewusst als schlanke statische Website aufgebaut und benötigt keinen Build-Schritt.

## Struktur

- `index.html` – Startseite, Karrierebereich und rechtliche Inhalte
- `styles.css` – Designsystem, Layout und responsive Darstellung
- `script.js` – Navigation, Animationen und Kontaktformular
- `assets/` – lokale Grafiken und Icons
- `.nojekyll` – direkte, unveränderte Veröffentlichung über GitHub Pages

## Lokal ansehen

Die Website über einen lokalen Webserver öffnen, zum Beispiel auf Port 4173. Ein Doppelklick auf `index.html` funktioniert größtenteils ebenfalls, ist für Tests aber nicht empfohlen.

## Vor Veröffentlichung prüfen

- Impressum und Datenschutzerklärung sind direkt in die Website integriert.
- Das Kontaktformular öffnet das E-Mail-Programm des Besuchers. Für direkten Versand ist später ein Formular-Endpunkt oder Backend nötig.
- Exakte Einsatzgebiete in NRW können ergänzt werden, sobald sie feststehen.
- Die Website ist für eine Veröffentlichung über GitHub Pages vorbereitet.
